BezelHUD is a Quicksilver Interface.

It was originally created by Julius Eckert. I updated it to version 1.2a2, making it compatible with Snow Leopard.

At the current time it only works with the newest builds of QuickSilver (b54a7). Also, there is an issue surrounding Spaces which should be resolved shortly.
