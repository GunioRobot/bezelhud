BezelHUD is a Quicksilver Interface originally created by Julius Eckert.

The current version is 1.2a4. It plays well with Snow Leopard and Spaces as well as the the newest versions of QuickSilver (ß57).

You can download the regular version or a version without the blurred background. Also, if you’re interested, check out the source. Please report any bugs on GitHub.
